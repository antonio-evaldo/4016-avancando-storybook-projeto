# Importando o componente Button

Importe o botão da seguinte forma no seu projeto:

```ts
import { ButtonComponent } from '@alfabit-alura-teste/button';
```

Este é o código Angular para utilizar no template:

<Source of={ButtonStories.PrimaryButton} />

<Canvas of={ButtonStories.PrimaryButton} layout="centered" sourceState="none" />
